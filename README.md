# Cara Pakainya:
1. buka file ini di vs code.
2. lalu buka terminal ketik npm install dan npx expo install --fix.
3. lalu lanjutkan ketik npx expo start dan akan muncul QR.
4. lalu buka Expo Go di smartphone kalian dan scan.
5. Selesai Selamat Mencoba.
